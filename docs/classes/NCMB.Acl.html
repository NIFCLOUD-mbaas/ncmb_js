NCMB.Acl Class

Defined in: lib/acl.js:5

オブジェクトへのアクセス権限を設定するクラスです。

DataStoreやUserクラスのインスタンスに付加して利用します。
指定するユーザおよびロールはmobile backendに保存済みである必要があります。
