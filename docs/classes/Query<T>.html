Query<T> Class

Defined in: lib/query.js:6

オブジェクトの検索を行うモジュールです。

DataStore, User, Role, Fileクラスから呼び出し、それぞれのクラスメソッドとして利用します。 検索条件を設定するメソッドに続けてfetch/fetchAllをメソッドチェーンで実行して利用します。
